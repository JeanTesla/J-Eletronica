O projeto permite um simples gerenciamento de garantia e fluxo de eletrônicos. 
Sendo possível informar: <br>
-> Data de entrada <br>
-> Data de saída <br>
-> Gerar termo de garantia com assinatura no formato PDF <br>
-> Nome do equipamento <br>
-> Defeito <br>
